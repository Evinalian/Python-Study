## 1. 字典是什么 —— 用"寻址"来理解哈希表

### 1.1 类比：快递柜

想象你有一个快递柜，每个包裹上贴着收件人的名字（key），柜子里放着包裹内容（value）。

- 你要取"张三"的包裹 → 你不需要逐个翻找所有柜子，而是通过"张三"这个名字直接定位到对应的柜子
- 这就是字典的核心能力：**O(1) 的平均查找时间**

在底层，Python 的 dict 使用**哈希表**（hash table）实现：

1. 对 key 调用 `hash()` 函数得到一个整数（哈希值）
2. 用这个哈希值计算出 key 在内部数组中的位置（索引）
3. 如果该位置已被别的 key 占用（哈希冲突），用探测序列找到下一个空位
4. 在该位置存储 key-value 对

从 Python 3.7 开始，dict **保证按照插入顺序**迭代。所以：

```python
d = {"z": 1, "a": 2, "m": 3}
print(list(d.keys()))   # ['z', 'a', 'm'] —— 按插入顺序，不是字母顺序！
```

### 1.2 key 必须满足的条件

dict 的 key 必须同时满足两个条件：

1. **可哈希**（定义了 `__hash__` 方法）：能够计算出一个在 key 生命周期内不变的哈希值
2. **可比较相等**（定义了 `__eq__` 方法）：当哈希冲突时，判断两个 key 是否真的相同

```python
# 不可变类型 → 可哈希
d = {
    "name": "张三",        # str 可哈希
    42: "答案",            # int 可哈希
    (1, 2): "坐标",        # tuple 可哈希
}

# 可变类型 → 不可哈希
# d = {[1, 2]: "列表"}     # TypeError: unhashable type: 'list'
# d = {{1, 2}: "集合"}     # TypeError: unhashable type: 'set'
```

**hash 约定**：如果 `a == b`，则 `hash(a) == hash(b)`。反之不一定成立（哈希冲突）。

## 2. 字典基本操作

### 2.1 增删改查

```python
d = {"name": "张三", "age": 20}

# === 增 / 改 ===
d["score"] = 95       # key 不存在 → 新增
d["age"] = 21         # key 已存在 → 覆盖

# === 查 ===
print(d["name"])      # "张三"
# print(d["gender"])  # KeyError! —— key 不存在时直接索引会抛异常

# === 删 ===
d.pop("score")        # 删除并返回 95
del d["name"]         # 直接删除
d.clear()             # 清空整个字典
```

### 2.2 get vs [] —— 安全的取值方式

这是新手最容易忽略的区别：

```python
d = {"name": "张三", "age": 20}

# 方式一：[] 直接索引 —— key 不存在时会报错
# print(d["gender"])       # KeyError: 'gender'

# 方式二：get() —— key 不存在时返回 None（或你指定的默认值）
print(d.get("gender"))          # None —— 不报错
print(d.get("gender", "未知"))  # "未知" —— 自定义默认值
print(d.get("name", "未知"))    # "张三" —— key 存在时正常返回
```

**实践建议**：
- 如果确定 key 一定存在，用 `d[key]`（更直接）
- 如果不确定 key 是否存在，用 `d.get(key, default)`（更安全）

### 2.3 遍历字典的多种方式

```python
d = {"name": "张三", "age": 20, "city": "北京"}

# 遍历 key（默认行为）
for key in d:
    print(key)

# 遍历 value
for val in d.values():
    print(val)

# 遍历 key-value 对（最常用）
for key, val in d.items():
    print(f"{key} → {val}")
```

### 2.4 字典推导式

```python
# 把列表转成字典
keys = ["name", "age", "city"]
values = ["张三", 20, "北京"]
person = {k: v for k, v in zip(keys, values)}
print(person)  # {'name': '张三', 'age': 20, 'city': '北京'}

# 键值互换
d = {"a": 1, "b": 2}
swapped = {v: k for k, v in d.items()}
print(swapped)  # {1: 'a', 2: 'b'}

# 过滤
scores = {"张三": 95, "李四": 55, "王五": 88}
passed = {name: score for name, score in scores.items() if score >= 60}
print(passed)  # {'张三': 95, '王五': 88}
```

## 3. defaultdict —— 不用每次检查 key 是否存在

### 3.1 问题场景：分组

```python
# 有一个列表，想把相同 key 的值收集到一起
data = [("a", 1), ("b", 2), ("a", 3), ("c", 4), ("b", 5)]

# 不用 defaultdict 的写法：每次都要检查
result = {}
for key, val in data:
    if key not in result:
        result[key] = []     # 手动创建空列表
    result[key].append(val)
print(result)  # {'a': [1, 3], 'b': [2, 5], 'c': [4]}

# 用 setdefault：稍好但每次调用都创建空列表
result = {}
for key, val in data:
    result.setdefault(key, []).append(val)
```

### 3.2 defaultdict 的优雅写法

```python
from collections import defaultdict

# list 在这里是"工厂函数"——当 key 不存在时，调用 list() 产生一个空列表
result = defaultdict(list)     # 注意：传的是 list，不是 list()！

for key, val in data:
    result[key].append(val)    # 不需要检查！key 不存在时自动创建 []

print(dict(result))  # {'a': [1, 3], 'b': [2, 5], 'c': [4]}
```

### 3.3 为什么传 `int` 而不是 `int()`

```python
# defaultdict 参数要求是一个"可调用对象"（函数/类），
# 它会在 key 缺失时被调用，不传参数，用返回值作为默认值

# int —— 这是一个类（可调用对象）
# int() —— 这调用了 int 类，得到 0（一个整数，不可调用）

# 正确：传 int（类本身）
counter = defaultdict(int)    # 缺省值为 int() = 0
counter["apple"] += 1
print(counter["apple"])       # 1

# 正确：传 list（类本身）
groups = defaultdict(list)    # 缺省值为 list() = []
groups["fruits"].append("apple")

# 错误：
# d = defaultdict(int())     # TypeError: first argument must be callable or None
#                            # int() 返回的 0 是不可调用的
```

常用的工厂函数：

| 工厂函数 | 缺省值 | 用途 |
|----------|--------|------|
| `int` | `0` | 计数器 |
| `list` | `[]` | 分组 |
| `set` | `set()` | 去重分组 |
| `str` | `""` | 字符串拼接 |
| `lambda: "默认值"` | `"默认值"` | 自定义默认值 |

## 4. Counter —— 统计计数的神器

### 4.1 基本用法

```python
from collections import Counter

# 从可迭代对象直接创建
words = ["apple", "banana", "apple", "orange", "banana", "apple"]
cnt = Counter(words)
print(cnt)  # Counter({'apple': 3, 'banana': 2, 'orange': 1})

# 从字符串创建（统计字符频率）
cnt = Counter("abracadabra")
print(cnt)  # Counter({'a': 5, 'b': 2, 'r': 2, 'c': 1, 'd': 1})
```

### 4.2 most_common —— 最常见的那几个

```python
text = """
Python is a programming language. Python is used for data science,
AI, web development. Python is easy to learn but powerful.
"""
import re
words = re.findall(r"\b[a-zA-Z]+\b", text.lower())
cnt = Counter(words)
print(cnt.most_common(3))
# [('python', 3), ('is', 3), ('a', 1)]
```

### 4.3 Counter 的数学运算

```python
c1 = Counter("abc")
c2 = Counter("bcd")

print(c1 + c2)    # Counter({'b': 2, 'c': 2, 'a': 1, 'd': 1})  —— 求和
print(c1 - c2)    # Counter({'a': 1})                          —— 减法（过滤非正值）
print(c1 & c2)    # Counter({'b': 1, 'c': 1})                  —— 取最小值（交集）
print(c1 | c2)    # Counter({'a': 1, 'b': 1, 'c': 1, 'd': 1}) —— 取最大值（并集）
```

## 5. set —— 没有 value 的字典

### 5.1 set 底层也是哈希表

`set` 的底层实现和 `dict` 基本相同——可以理解为"只有 key 没有 value 的 dict"。因此：
- O(1) 平均成员检测
- 元素必须可哈希（和 dict key 的要求一样）
- 自动去重

### 5.2 创建和基本操作

```python
# 创建
s = {1, 2, 3}             # 字面量
s = set([1, 2, 2, 3, 3])  # 从可迭代对象创建，重复自动去除 → {1, 2, 3}
empty = set()              # 空集合：{} 是空字典！

# 增删查
s = {1, 2}
s.add(3)                   # {1, 2, 3} —— 添加
s.discard(2)               # {1, 3} —— 安全删除（不存在也不报错）
s.remove(1)                # {3} —— 删除（不存在抛 KeyError）
print(3 in s)              # True —— O(1) 成员检测
```

### 5.3 一行去重

```python
# 列表去重（不保留顺序）
lst = [3, 1, 2, 1, 3, 2, 4]
unique = list(set(lst))
print(unique)  # [1, 2, 3, 4] —— 但不保证顺序

# 列表去重（保留顺序）
unique_ordered = list(dict.fromkeys(lst))
print(unique_ordered)  # [3, 1, 2, 4] —— Python 3.7+ dict 保持插入顺序
```

### 5.4 集合运算

```python
a = {1, 2, 3, 4}
b = {3, 4, 5, 6}

# 并集：a 或 b 中的所有元素
print(a | b)       # {1, 2, 3, 4, 5, 6}
print(a.union(b))  # 同上

# 交集：同时在 a 和 b 中的元素
print(a & b)               # {3, 4}
print(a.intersection(b))   # 同上

# 差集：在 a 但不在 b 中的元素
print(a - b)            # {1, 2}
print(a.difference(b))  # 同上

# 对称差：只在 a 或只在 b 中的元素（排除两者共有）
print(a ^ b)                       # {1, 2, 5, 6}
print(a.symmetric_difference(b))   # 同上
```

### 5.5 子集和超集

```python
print({1, 2} <= {1, 2, 3})   # True —— 子集
print({1, 2, 3} >= {1, 2})   # True —— 超集
print({1, 2} < {1, 2})       # False —— 真子集（排除相等）
```

### 练习 2：统计字符出现次数

写一个函数 `char_count(s)`，用字典统计字符串中每个字符的出现次数（区分大小写）。

**参考答案**：

```python
def char_count(s):
    """统计每个字符出现次数"""
    counts = {}
    for ch in s:
        if ch in counts:
            counts[ch] += 1
        else:
            counts[ch] = 1
    return counts

# 测试
result = char_count("hello")
print(result)  # {'h': 1, 'e': 1, 'l': 2, 'o': 1}

# 更优雅的写法：用 get 设置默认值
def char_count_v2(s):
    counts = {}
    for ch in s:
        counts[ch] = counts.get(ch, 0) + 1
    return counts
```

### 练习 4：用 defaultdict 实现倒排索引

倒排索引（Inverted Index）是搜索引擎的核心数据结构：给定一些文档，对每个词，记录它出现在哪些文档中。

```python
# 输入：文档字典 {doc_id: text}
docs = {
    1: "Python is great for data science",
    2: "Python is also used for web development",
    3: "Java is popular for enterprise applications",
}

# 期望输出（倒排索引）：
# {
#     "python": {1, 2},
#     "is": {1, 2, 3},
#     "great": {1},
#     ...
# }
```

**参考答案**：

```python
from collections import defaultdict

def build_inverted_index(docs):
    """
    构建倒排索引：词 → 包含该词的所有文档 ID 集合

    参数:
        docs: dict，键为 doc_id，值为文档文本

    返回:
        defaultdict，键为词（小写），值为 doc_id 集合
    """
    index = defaultdict(set)  # 缺省值是一个空 set

    for doc_id, text in docs.items():
        # 分词：转小写后按空白拆分（简化处理）
        words = text.lower().split()
        for word in words:
            index[word].add(doc_id)  # key 不存在时自动创建 set()

    return index


# 测试
docs = {
    1: "Python is great for data science",
    2: "Python is also used for web development",
    3: "Java is popular for enterprise applications",
}

index = build_inverted_index(docs)

# 查询：搜索 "python"
query = "python"
if query in index:
    print(f"'{query}' 出现在文档: {index[query]}")

# 打印完整倒排索引
for word, doc_ids in sorted(index.items()):
    print(f"{word:15s} → {doc_ids}")

# 输出示例：
# applications    → {3}
# ...
# is              → {1, 2, 3}
# python          → {1, 2}
# science         → {1}
# ...
```

## 常见错误

### 错误 1：`{}` 是空字典，不是空集合

```python
s = {}           # <class 'dict'>
print(type(s))   # <class 'dict'>

s = set()        # <class 'set'> —— 空集合只能用 set()
print(type(s))   # <class 'set'>
```

### 错误 2：用不存在的 key 直接索引

```python
d = {"a": 1}
# print(d["b"])         # KeyError!

# 正确做法：用 get
print(d.get("b"))       # None
print(d.get("b", 0))    # 0
```

### 错误 3：遍历字典时修改它

```python
d = {"a": 1, "b": 2, "c": 3}
# for k in d:           # RuntimeError: dictionary changed size during iteration
#     if d[k] < 2:
#         del d[k]

# 正确做法：遍历 key 的快照
for k in list(d.keys()):
    if d[k] < 2:
        del d[k]
print(d)                # {'b': 2, 'c': 3}
```

### 错误 4：用可变对象作为 dict key

```python
# d = {[1, 2]: "value"}    # TypeError: unhashable type: 'list'
d = {(1, 2): "value"}      # OK —— tuple 不可变、可哈希
```

### 错误 5：set 的 remove 对不存在的元素抛异常

```python
s = {1, 2, 3}
s.remove(4)           # KeyError: 4

# 用 discard 安全删除（不存在也不报错）
s.discard(4)          # 静默忽略
```

### 错误 6：defaultdict 参数传错

```python
from collections import defaultdict

# 错误：传的是 int() 的结果 0，而 0 不是可调用对象
# d = defaultdict(0)               # TypeError

# 正确：传 int 类本身（它是可调用对象）
d = defaultdict(int)               # 缺省值 0
```

## 本章小结

- **dict** 底层是哈希表，O(1) 平均查找；Python 3.7+ 保证插入顺序
- **key 必须可哈希**（定义了 `__hash__` 且 `__eq__`），且哈希值应在生命周期内不变
- **取值的四种方式**：`d[key]`（确定存在）、`d.get(key, default)`（安全读）、`d.setdefault(key, default)`（不存在时设置）、`defaultdict(factory)`（自动创建缺省值）
- **defaultdict(factory)**：factory 必须是可调用对象（如 `int`、`list`、`set`），缺 key 时自动调用
- **Counter**：`most_common(n)` 取最高频、支持 `+-&|` 数学运算
- **set** 是只有 key 的 dict，O(1) 成员检测和去重；`frozenset` 是其不可变版本
- **集合运算**：`| & - ^` 及对应的 `update` 系列方法；`<= >=` 判断子集和超集
